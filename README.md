# css-file-hosting
Easy css file online editor and hosting
Demo: https://hosting.wiktor.tech/

### Configuration (config.json)

```json
{
    "port": 3000,
    "id_lenght": 10,
    "base_url": "http://localhost:3000/",
    "app_name": "Hosting plikow CSS1"
}
```

### Run app

```bash
npm install
npm start
```
